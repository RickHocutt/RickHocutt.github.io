About Me

I am a Front End Web Developer with sixteen years of graphic design experience and six years of experience building websites that conform to current web standards. I hand code valid HTML 5, CSS 3, and JavaScript.

I have worked as a Designer and Front End Web Developer for LEGOLAND, AutoZone, Cardinal Health and Aon. My duties have included developing user flows and application interfaces for AutoZone.com, AutoZone.com.br, UPoint . On my own time I enjoy building sites using Drupal, Wordpress, Jekyll, Grunt and a slew of other Front-End tools.

This site is hosted on GitHub, want to see the code? Check out the repo at: https://github.com/RickHocutt/RickHocutt.github.io.
